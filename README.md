# Metallic Glass vs Non-Metallic Glass 분류 (대학교 프로젝트)

대학교 수업/프로젝트에서 수행한 **금속 유리(Metallic Glass)** 와 **비금속 유리(Non-Metallic Glass)** 를 구별하는 **이진 분류(Classification)** 과제입니다.

## 목표
- 입력 데이터(특성값)를 기반으로 시료가 **MG / NMG** 중 무엇인지 예측
- 데이터 탐색(EDA) 및 간단한 모델링을 통해 분류 성능 확인
